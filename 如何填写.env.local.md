# 如何填写 .env.local 文件

## 📝 步骤说明

### 1. 打开 .env.local 文件

**方法 A：使用 VS Code（推荐）**
- 在 VS Code 中，点击左侧文件浏览器
- 找到 `.env.local` 文件
- 双击打开

**方法 B：使用记事本**
- 在文件管理器中，进入项目文件夹
- 找到 `.env.local` 文件
- 右键点击 → 打开方式 → 记事本

**方法 C：使用 Cursor**
- 在 Cursor 中，按 `Ctrl+P` 打开文件
- 输入 `.env.local` 并回车

### 2. 填写 Supabase URL

找到这一行：
```
VITE_SUPABASE_URL=https://your-project-id.supabase.co
```

替换为你的实际 URL：
```
VITE_SUPABASE_URL=https://你的项目ID.supabase.co
```

**示例：**
```
VITE_SUPABASE_URL=https://abcdefghijklmnop.supabase.co
```

### 3. 填写 anon key

找到这一行：
```
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

替换为你的实际 anon key：
```
VITE_SUPABASE_ANON_KEY=你的完整anon密钥
```

**示例：**
```
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImFiY2RlZmdoaWprbG1ub3AiLCJyb2xlIjoiYW5vbiIsImlhdCI6MTYzODk2NzI5MCwiZXhwIjoxOTU0NTQzMjkwfQ.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 4. 保存文件

- 按 `Ctrl+S` 保存
- 或点击文件菜单 → 保存

### 5. 验证格式

确保文件格式正确：

✅ **正确格式：**
```
VITE_SUPABASE_URL=https://abcdefghijklmnop.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

❌ **错误格式：**
```
VITE_SUPABASE_URL="https://..."  # 不要加引号
VITE_SUPABASE_ANON_KEY = "..."   # 不要有空格在等号两边
```

## 📋 完整示例

```env
# Supabase Configuration
VITE_SUPABASE_URL=https://abcdefghijklmnop.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImFiY2RlZmdoaWprbG1ub3AiLCJyb2xlIjoiYW5vbiIsImlhdCI6MTYzODk2NzI5MCwiZXhwIjoxOTU0NTQzMjkwfQ.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## ✅ 检查清单

填写完成后，检查：

- [ ] URL 以 `https://` 开头
- [ ] URL 以 `.supabase.co` 结尾
- [ ] anon key 以 `eyJ` 开头（很长，200+ 字符）
- [ ] 等号两边没有空格
- [ ] 没有添加引号
- [ ] 文件已保存
