# 如何找到 .env.local 文件

## 🔍 方法 1：在 Cursor 中搜索（最简单）

1. **按 `Ctrl+Shift+F`** 打开全局搜索
2. **在搜索框输入：** `.env.local`
3. **如果文件存在，会显示出来**

## 🔍 方法 2：使用文件管理器

1. **打开文件管理器（Windows 资源管理器）**
2. **导航到项目文件夹：**
   ```
   C:\Users\吴昊天\Desktop\MSBC网站开发\msbc-website
   ```
3. **在文件管理器顶部，点击"查看"标签**
4. **勾选"隐藏的项目"**（这样可以看到以点开头的文件）
5. **查找 `.env.local` 文件**

## 🔍 方法 3：直接打开文件

如果文件存在但搜不到，可以：

1. **在 Cursor 中按 `Ctrl+P`**
2. **输入完整路径：**
   ```
   .env.local
   ```
3. **或者输入：**
   ```
   env.local
   ```
   （可能文件名没有前面的点）

## ✅ 如果文件不存在，重新创建

### 最简单的方法：

1. **在 Cursor 中，找到 `env.local.template` 文件**
2. **右键点击它**
3. **选择 "Copy"（复制）**
4. **在空白处右键，选择 "Paste"（粘贴）**
5. **重命名新文件为 `.env.local`**

### 或者手动创建：

1. **在 Cursor 中按 `Ctrl+N` 创建新文件**
2. **粘贴以下内容：**
   ```
   VITE_SUPABASE_URL=https://your-project-id.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key-here
   ```
3. **按 `Ctrl+S` 保存**
4. **文件名输入：** `.env.local`
5. **保存位置：项目根目录（与 package.json 同级）**

## 🎯 验证文件位置

文件应该在以下位置：
```
C:\Users\吴昊天\Desktop\MSBC网站开发\msbc-website\.env.local
```

与这些文件在同一目录：
- `package.json`
- `vite.config.ts`
- `env.local.template`

## 💡 提示

如果文件创建后还是搜不到：
- 可能是 Cursor 的索引还没更新
- 尝试重启 Cursor
- 或者直接在文件管理器中打开文件编辑
