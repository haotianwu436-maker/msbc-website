### 步骤 2: 部署到 Vercel（10分钟）

#### 2.1 登录 Vercel
1. 访问：https://vercel.com
2. 点击 `Sign Up` 或 `Log In`
3. 选择 `Continue with GitHub`（使用 GitHub 账号登录）

#### 2.2 导入项目
1. 点击 `Add New Project`
2. 选择你的仓库：`haotianwu436-maker/msbc-website`
3. 点击 `Import`

#### 2.3 配置项目
**Build Settings**（通常自动检测）：
- Framework Preset: `Other`
- Build Command: `pnpm build`
- Output Directory: `dist/public`
- Install Command: `pnpm install`

**Environment Variables**（必须配置）：
1. 点击 `Environment Variables`
2. 添加第一个变量：
   ```
   Name: VITE_SUPABASE_URL
   Value: （从 .env.local 复制你的 Supabase URL）
   ```
3. 添加第二个变量：
   ```
   Name: VITE_SUPABASE_ANON_KEY
   Value: （从 .env.local 复制你的 Anon Key）
   ```
4. 选择所有环境：`Production`, `Preview`, `Development`（全选）
5. 点击 `Add` 保存每个变量

#### 2.4 部署
1. 点击 `Deploy`
2. 等待部署完成（通常 1-3 分钟）
3. 部署完成后，你会得到一个临时域名：`https://msbc-website-xxx.vercel.app`

#### 2.5 测试
1. 访问临时域名，确认网站正常
2. 访问 `/admin`，测试登录
3. 测试添加/修改数据功能

---

### 步骤 3: 购买并配置域名（15分钟）

#### 3.1 购买域名

**推荐域名注册商**：
- **Namecheap**（推荐）：https://www.namecheap.com
- **Cloudflare**：https://www.cloudflare.com/products/registrar/

**购买步骤**（以 Namecheap 为例）：
1. 访问 Namecheap
2. 搜索你想要的域名（例如：`msbc2026.com`）
3. 添加到购物车
4. 选择购买年限（建议 1-2 年）
5. 添加隐私保护（Whois Privacy，通常免费）
6. 结账付款

**域名建议**：
- `msbc2026.com`
- `msbc.org`
- `msbc.my`（马来西亚域名）

#### 3.2 在 Vercel 配置域名

1. **添加域名**：
   - 在 Vercel Dashboard，进入你的项目
   - 点击 `Settings` → `Domains`
   - 输入你的域名（例如：`msbc2026.com`）
   - 点击 `Add`

2. **配置 DNS**：
   - Vercel 会显示需要配置的 DNS 记录
   - 通常需要添加：
     ```
     Type: CNAME
     Name: @
     Value: cname.vercel-dns.com（Vercel 会提供具体值）
     ```

3. **在域名注册商配置 DNS**：
   - 登录 Namecheap
   - 进入 `Domain List` → 选择你的域名
   - 点击 `Advanced DNS`
   - 添加 DNS 记录（按照 Vercel 的提示）
   - 保存

4. **等待 DNS 生效**：
   - 通常 5-30 分钟
   - 最长可能需要 24-48 小时
   - 可以在 https://dnschecker.org 检查 DNS 传播状态

5. **验证**：
   - DNS 生效后，Vercel 会自动验证
   - 验证成功后，访问 `https://你的域名.com`，确认网站正常

## 📞 遇到问题？

### 部署失败
- 检查环境变量是否配置
- 检查构建日志错误
- 确认 GitHub 仓库连接正常

### DNS 不生效
- 等待更长时间（最长 48 小时）
- 使用 https://dnschecker.org 检查
- 确认 DNS 记录配置正确

### 网站无法访问
- 检查环境变量是否正确
- 检查浏览器控制台错误（F12）
- 确认 Supabase 项目正常运行
