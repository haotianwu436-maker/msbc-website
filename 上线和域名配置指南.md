## 🚀 第二步：部署网站

### 方案 A: 使用 Vercel（推荐）

#### 步骤 1: 连接 GitHub

1. **登录 Vercel**
   - 访问：https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 `Add New Project`
   - 选择你的 GitHub 仓库（`msbc-website`）
   - 点击 `Import`

#### 步骤 2: 配置项目

**Build Settings**（通常自动检测）:
- Framework Preset: `Other`
- Build Command: `pnpm build`
- Output Directory: `dist/public`
- Install Command: `pnpm install`

**Environment Variables**（必须配置）:
- 点击 `Environment Variables`
- 添加：
  ```
  VITE_SUPABASE_URL = https://你的项目.supabase.co
  VITE_SUPABASE_ANON_KEY = 你的 anon key
  ```
- 选择所有环境（Production, Preview, Development）
- 点击 `Save`

#### 步骤 3: 部署

1. 点击 `Deploy`
2. 等待部署完成（通常 1-3 分钟）
3. 部署完成后，你会得到一个临时域名：`https://你的项目名.vercel.app`

#### 步骤 4: 测试

1. 访问临时域名，确认网站正常
2. 访问 `/admin`，测试登录
3. 测试添加/修改数据功能

---

### 方案 B: 使用 Netlify

#### 步骤 1: 连接 GitHub

1. **登录 Netlify**
   - 访问：https://netlify.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 `Add new site` → `Import an existing project`
   - 选择你的 GitHub 仓库
   - 点击 `Import`

#### 步骤 2: 配置项目

**Build settings**:
- Build command: `pnpm build`
- Publish directory: `dist/public`

**Environment variables**（必须配置）:
- 点击 `Site settings` → `Environment variables`
- 添加：
  ```
  VITE_SUPABASE_URL = https://你的项目.supabase.co
  VITE_SUPABASE_ANON_KEY = 你的 anon key
  ```
- 点击 `Save`

#### 步骤 3: 部署

1. 点击 `Deploy site`
2. 等待部署完成
3. 你会得到一个临时域名：`https://随机名称.netlify.app`

### 购买域名步骤（以 Namecheap 为例）

#### 步骤 1: 注册账号

1. 访问：https://www.namecheap.com
2. 点击 `Sign Up` 注册账号
3. 验证邮箱

#### 步骤 2: 搜索域名

1. 在首页搜索框输入你想要的域名
   - 例如：`msbc2026.com`、`msbc.org`、`msbc.my`（马来西亚域名）
2. 查看可用性和价格
3. 选择合适的域名

#### 步骤 3: 购买

1. 点击 `Add to Cart`
2. 选择购买年限（建议 1-2 年）
3. 添加隐私保护（Whois Privacy，通常免费或很便宜）
4. 结账付款

#### 步骤 4: 等待激活

- 通常几分钟内激活
- 收到确认邮件

## 🔗 第四步：配置域名到部署平台

### 方案 A: Vercel 配置域名

#### 步骤 1: 添加域名

1. 在 Vercel Dashboard，进入你的项目
2. 点击 `Settings` → `Domains`
3. 输入你的域名（例如：`msbc2026.com`）
4. 点击 `Add`

#### 步骤 2: 配置 DNS

Vercel 会显示需要配置的 DNS 记录：

**通常需要添加**:
- **A Record** 或 **CNAME Record**
  - 类型：`CNAME`
  - 名称：`@` 或 `www`
  - 值：`cname.vercel-dns.com`（Vercel 会提供具体值）

**示例 DNS 配置**:
```
Type: CNAME
Name: @
Value: cname.vercel-dns.com
```

#### 步骤 3: 在域名注册商配置 DNS

**Namecheap**:
1. 登录 Namecheap
2. 进入 `Domain List` → 选择你的域名
3. 点击 `Advanced DNS`
4. 添加 DNS 记录（按照 Vercel 的提示）
5. 保存

**等待 DNS 生效**:
- 通常 5-30 分钟
- 最长可能需要 24-48 小时

#### 步骤 4: 验证

1. 等待 DNS 生效后，Vercel 会自动验证
2. 验证成功后，域名就可以访问了
3. 访问 `https://你的域名.com`，确认网站正常

---

### 方案 B: Netlify 配置域名

#### 步骤 1: 添加域名

1. 在 Netlify Dashboard，进入你的站点
2. 点击 `Domain settings` → `Add custom domain`
3. 输入你的域名
4. 点击 `Verify`

#### 步骤 2: 配置 DNS

Netlify 会显示需要配置的 DNS 记录：

**通常需要添加**:
- **A Record**:
  - 类型：`A`
  - 名称：`@`
  - 值：`75.2.60.5`（Netlify 会提供具体 IP）

- **CNAME Record**（可选，用于 www）:
  - 类型：`CNAME`
  - 名称：`www`
  - 值：`你的站点名.netlify.app`

#### 步骤 3: 在域名注册商配置 DNS

按照 Netlify 的提示，在域名注册商添加 DNS 记录。

## 📞 遇到问题？

### 部署问题
- 检查环境变量是否配置
- 检查构建日志是否有错误
- 检查 GitHub 仓库是否正确连接

### DNS 问题
- 检查 DNS 记录是否正确配置
- 使用 https://dnschecker.org 检查 DNS 传播
- 等待更长时间（最长 48 小时）

### 域名问题
- 联系域名注册商客服
- 检查域名是否已激活
- 检查域名是否已过期
